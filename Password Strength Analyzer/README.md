Password Strength Analyzer

Objective

This project is used to analyze the strength of user passwords and provide suggestions to improve weak passwords.

Features

* Checks password length
* Detects uppercase and lowercase letters
* Checks numbers and special characters
* Displays password strength:

  * Weak
  * Medium
  * Strong
* Provides feedback for improvement

 Technologies Used

 Python
 Regular Expressions

Project Files

password_analyzer.py → Main Python program
Password Strength Analyzer Report → Project report/documentation
Password Strength Analyzer output → Output screenshots

How to Run the Project

1. Install Python
2. Git Hub link https://github.com/giona-jasley/password-strength-analyzer/tree/main/Password%20Strength%20Analyzer
3. Download the project files
4. Open terminal in project folder

5. Run:

python password_analyzer.py

Sample Output

Example:

Enter your password: Hello@123

Password Strength: STRONG
